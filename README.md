# handwriting-recognition
An implementation of K-Nearest-Neighbors classification to recognize handwritten digits.

## Usage
Download the program:
```bash
git clone https://github.com/gmcgehee/handwriting-recognition.git
```

Run the program:
```bash
python3 handwriting_recognition.py 
```
Draw a digit 0-9 in the box that appears.

## Dependencies
`customtkinter` for a clean GUI, `numpy` for data processing, `joblib` to load the stored model, `PIL` for image processing, `matplotlib` for image display.

```bash
pip install customtkinter Pillow numpy joblib matplotlib
```
